# Security Policy

## Supported versions

Only the latest version on the `main` branch receives security fixes.

## Reporting a vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report security issues by emailing the maintainer directly or by using [GitHub's private vulnerability reporting](https://github.com/TuroYT/Armature/security/advisories/new).

Include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You will receive a response within 72 hours. Once the vulnerability is confirmed and patched, a public advisory will be published.
